## 🚀 Быстрый старт

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/DOLLDI/Lash-master-site.git
   cd Lash-master-site
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите сервер:
   ```bash
   node server.js
   ```
4. Откройте сайт:
   - Главная: [http://localhost:3000](http://localhost:3000)
   - Админ-панель: [http://localhost:3000/admin.html](http://localhost:3000/admin.html)

- `styles.css`, `admin.css` — Стили

- `data.json` — Данные сайта
